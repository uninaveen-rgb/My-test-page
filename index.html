Tips
  • Use the search box to find topics quickly.
  • Choose a category (e.g., Colors, Shapes).
  • Add your own videos by editing the VIDEOS JSON.
How to add videos
  1. Find the VIDEOS array in the script.
  2. Add an item: { title, srcType, src, thumb, category, lang, tags, desc }.
  3. For YouTube, set srcType:"youtube" and src:"VIDEO_ID".
  4. For local MP4, set srcType:"mp4" and src:"media/yourfile.mp4".
